## Notes

- All documents are available as HTML (online) or PDF download via the Oracle Documentation site.
- To download as PDF: open the link above → click the **PDF** button in the top right.
- Oracle requires a free Oracle account to download PDFs.
- Claude Projects supports PDF uploads up to a certain size — split very large PDFs by chapter if needed.
